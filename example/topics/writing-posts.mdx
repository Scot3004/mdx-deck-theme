# Trabajando con posts

Para la creación de los posts hay que tener presente que puede hacerse creando páginas, sin embargo, al tener estilos similares es una buena idea crear un folder donde esten estilos
Es muy común usar Markdown para los posts, debido a su sintaxis simple, para lograrlo, se requiere transformar estos posts, ya sea a traves de plugins o bien crearlos programaticamente con el archivo `gatsby-node.js`
